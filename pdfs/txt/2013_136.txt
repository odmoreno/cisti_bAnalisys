Usability assessment of web interfaces: User Testing
ABSTRACT
The assessment of the usability aims to identify specific problems on the interface it is a process which uses several techniques and methods. One of those techniques the user testing is here shown through the completion of a set of tasks that aims to observe and measure the user's interaction with the interface. In the case presented the user Testing was built based on a sequence of interviews with real and representative users of the community which uses the Website. From this came the definition of several profiles, with the three most relevant profiles being posteriorly selected for one set of ten tasks. Three open questions were additionally included which, along the comments made by the participants during the user Testing permitted to obtain subjective information as it is the user's satisfaction assessment. The results obtained were very clear as they indicated that the Website's usability is quite reasonable, that is, in general, the users execute their tasks with ease, speed and satisfaction.
